 1. Template LaTeX untuk Doktor DTETI UGM ini disusun untuk:
   a) Ujian Komprehensif (UK)
   b) Seminar Hasil 1 (SH1)
   c) Seminar Hasil 2 (SH2)
   d) Penilaian Kelayakan Disertasi (PKD) dan Ujian Tertutup (UT)

2. Anda cukup bekerja dengan file [.tex]. Dengan konten/isi yang sama, template ini mempunyai file [.tex] yang berbeda berdasarkan kebutuhan masing-masing ujian (UK, SH1, SH2, PKD/UT). Hal ini ditujukan untuk memberi kemudahan dalam meneruskan dan mengembangkan konten/isi dari UK, menuju ke SH1, SH2, dan terakhir PKD/UT.

3. Template ini menyediakan fitur untuk unggah PDF, umumnya diperlukan pada lembar-lembar yang membutuhkan tanda tangan (TTD)/materai. PDF yang akan diunggah cukup diletakkan dalam folder attachment dengan penyesuaian nama file dengan perintah pemanggilan di file [.tex] terkait. Instruksi-instruksi yang yang diperlukan telah diberikan pada file [.tex]. Template ini juga memasukkan fungsi hyperlink pada daftar isi, daftar gambar, daftar tabel, sitasi, gambar, tabel, dan rumus matematika.

4. Perbedaan file [.tex] tersebut antara lain:
   a) UK mempunyai 4 versi Lembar Persetujuan yang bisa dipilih: (i) Sebelum UK, untuk meminta TTD persetujuan dari promotor, (ii) Sebelum UK, setelah mendapat TTD persetujuan dari promotor, dengan unggah PDF ke naskah LaTeX, untuk dikumpulkan ke DTETI sebagai syarat mendaftar UK, (iii) Setelah UK, setelah revisi, untuk meminta TTD persetujuan revisi dari promotor dan penguji, (iv) Setelah UK, setelah revisi, setelah mendapat TTD persetujuan revisi dari promotor dan penguji, dengan unggah PDF ke naskah LaTeX, untuk dikumpulkan ke DTETI.
   b) SH1 mempunyai 2 versi Lembar Persetujuan yang bisa dipilih: (i) Sebelum SH1, untuk meminta TTD persetujuan dari promotor, (ii) Sebelum SH1, setelah mendapat TTD persetujuan dari promotor, dengan unggah PDF ke naskah LaTeX, untuk dikumpulkan ke DTETI sebagai syarat mendaftar SH1.
   c) SH2 mempunyai 2 versi Lembar Persetujuan yang bisa dipilih: (i) Sebelum SH2, untuk meminta TTD persetujuan dari promotor, (ii) Sebelum SH2, setelah mendapat TTD persetujuan dari promotor, dengan unggah PDF ke naskah LaTeX, untuk dikumpulkan ke DTETI sebagai syarat mendaftar SH2.
   d) PKD/UT mempunyai 4 versi Lembar Persetujuan yang bisa dipilih: (i) Sebelum PKD/UT, untuk meminta TTD persetujuan dari promotor, (ii) Sebelum PKD/UT, setelah mendapat TTD persetujuan dari promotor, dengan unggah PDF ke naskah LaTeX, untuk dikumpulkan ke DTETI sebagai syarat mendaftar PKD/UT, (iii) Setelah PKD/UT, setelah revisi, untuk meminta TTD persetujuan revisi dari promotor dan penguji, (iv) Setelah PKD/UT, setelah revisi, setelah mendapat TTD persetujuan revisi dari promotor dan penguji, dengan unggah PDF ke naskah LaTeX, untuk dikumpulkan ke DTETI.
   e) PKD/UT mempunyai Pernyataan Promovendus (Originality Statement) dan Prakata, sedangkan UK, SH1, dan SH2 tidak. Pernyataan Promovendus (Originality Statement) dan Prakata pada PKD/UT mempunyai 2 pilihan, sebelum TTD/TTD-materai dan setelah TTD/TTD-materai.

5. Hapus tanda % untuk mengaktifkan baris program terkait, berikan tanda % untuk menonaktifkan. Baris program dengan tanda %%%% mengartikan bahwa baris program itu adalah instruksi/komentar sehingga tidak boleh diaktifkan.

6. Template Latex ini memberikan 2 pilihan dalam mengatur penggalan kata (hypenation): 
   (a) Dengan menggunakan fitur penggalan kata di LaTeX menggunakan Language Package Bahasa Indonesia. Namun beberapa kata masih terdapat kesalahan. Oleh karena itu, perlu beberapa setting tambahan dengan parameter yang sudah berhasil digunakan oleh penulis Template Latex ini ketika menulis Naskah Tesis Magister DTETI, Ujian Komprehensif, dan SH1. Beberapa kata perlu didefinisikan penggalan katanya (kurang lebih 10 kata).
   (b) Dengan mematikan fungsi penggalan kata. Ketika dilakukan, akan membuat naskah mirip dengan Microsoft Word yang tidak mengijinkan adanya penggalan kata. Namun, ini kurang direkomendasikan karena salah satu kelebihan penggunaan LaTeX adalah kemampuan penggalan katanya.

7. LaTeX sangat populer dalam penulisan naskah profesional karena konsistensi formatnya. Pengetahuan dan kemampuan tentang dasar-dasar bahasa pemrograman akan membantu, tetapi bukan merupakan kewajiban dalam menggunakan Template LaTeX dan menulis tulisan ilmiah.
